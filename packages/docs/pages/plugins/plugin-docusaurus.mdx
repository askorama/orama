# Docusaurus Plugin

The plugin-docusaurus plugin allows you to index the content of your Docusaurus websites and offer text search to your visitors.

## Installation

You can install the plugin using any major Node.js package manager:

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>```bash npm install @orama/plugin-docusaurus ```</Tab>
  <Tab>```bash yarn add @orama/plugin-docusaurus ```</Tab>
  <Tab>```bash pnpm add @orama/plugin-docusaurus ```</Tab>
</Tabs>

## Usage

Add the plugin to your `docusaurus.config.js`:

```js
{
  // ...
  plugins: ["@orama/plugin-docusaurus"],
  // ...
}
```
